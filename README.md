# 	Бешеная курица

Попробуй набрать наибольшее количество очков в этой нелегкой игре! Верь своей курице и она снесет тебе золотое яйцо!
Бонусы:
- замедление обьектов
- маркер направления движения
- +1 жизни

Игра написана на C# Unity3D
